# 诗笺 (ShiJian)

落墨成诗，抚笺成画。

诗笺是一款专为 macOS 设计的国风诗词图文排版与美学创作应用。采用 Swift 与原生 SwiftUI 构建，提供宣纸洒金纹理、古典朱丝界格、自制金石印章以及禅意水墨交互，帮助您将中国古典诗词排版成极具质感的唯美图片，支持高清无损导出及系统原生分享。

---

## 核心美学特性

* 纸张材质滤镜：基于固定种子的确定性伪随机数算法，动态渲染高精度宣纸植物纤维或冷洒金金箔，强度自由微调，支持超清无损平铺导出。
* 古典信笺界格：支持绘制朱丝栏与乌丝栏。外框采用外粗内细精美双线套框，栏格在 2 至 20 范围内等宽切分，完美支持横栏与竖栏排版。
* 国风传统色盘：精选六大传统矿物与植物雅色（朱砂、黛蓝、烟灰、鸭蛋青、苍黄、藕荷），鼠标悬停显示古典雅称，点按即时修改字色。
* 毛玻璃悬浮控制栏：选中排版元素时，在其下方弹出轻量级毛玻璃操作条，随拖动或缩放以弹簧阻尼曲线实现平滑物理跟随，提供字号微调、图层移位和快捷删除。
* 禅意水墨占位：画布完全为空时，自动隐入朱砂八栏格和淡金色大字“笺”的写意水墨画卷；一经写入文本，图案自动渐隐。
* 动态排版与印章：支持文字块横竖排、字号、颜色、对齐、倾斜、描边、背景条与阴影调节，支持长句动态折行。印章提供方印、圆印、圆方印与椭圆印四种形制，内置斑驳做旧算法。

## 安装与运行

1. 下载本仓库根目录下的 诗笺.dmg 磁盘镜像文件。
2. 双击打开 诗笺.dmg，将 诗笺 应用程序图标拖拽至 Applications (应用程序) 文件夹完成安装。

### macOS 运行提示 (绕过安全拦截的方法)

由于本应用为个人独立作品，未购买 Apple 开发者账号进行证书签名，首次在 macOS 上运行时，系统可能会拦截并提示“已损坏，打不开”或“来自未知的开发者”。

解决方法：
打开系统自带的 终端 (Terminal.app) 软件，复制并运行以下命令，然后回车，即可双击正常运行应用：

```bash
xattr -cr /Applications/诗笺.app
```
