# 诗笺 (ShiJian) - 小红书宣传方案

---

## 📖 项目简介

**诗笺**是一款专为 macOS 设计的国风诗词图文排版与美学创作应用。采用 Swift 与原生 SwiftUI 构建，让你将中国古典诗词排版成极具质感的唯美图片。

## 📝 完整文案 (可直接复制)

### 正文：

✨落墨成诗，抚笺成画✨

终于！耗时半年打造的国风诗词排版神器「诗笺」上线了！

作为一个热爱传统文化的开发者，我一直想做一款能让诗词之美数字化呈现的工具。

🎨 **核心功能**：
▫️ 宣纸材质滤镜 - 动态渲染高精度宣纸植物纤维
▫️ 古典信笺界格 - 朱丝栏/乌丝栏，外粗内细精美双线套框
▫️ 六大传统色盘 - 朱砂、黛蓝、烟灰、鸭蛋青、苍黄、藕荷
▫️ 毛玻璃悬浮控制栏 - 优雅的交互体验
▫️ 禅意水墨占位 - 空画布时隐入写意水墨画卷
▫️ 金石印章系统 - 方印/圆印/圆方印/椭圆印四种形制

🎨 **12款渐变主题**：
夜墨金书｜古卷宣纸｜烟雨江南｜水墨山水
深林幽径｜落霞孤鹜｜青苔竹露｜黛蓝微雨
天青过雨｜胭脂晚妆｜落梅残雪｜晓风残月
